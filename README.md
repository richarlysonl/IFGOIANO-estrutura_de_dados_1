# Estrutura de Dados I
A disciplina de Estrutura de Dados I é fundamental para o desenvolvimento de algoritmos eficientes e para a compreensão da organização da informação na memória do computador. O curso foca no estudo de estruturas lineares e na análise de sua complexidade.

Conteúdo Principal: Estudo de Ponteiros e Alocação Dinâmica de Memória.

Estruturas Lineares: Implementação e manipulação de Listas Encadeadas (simples e duplas), Pilhas (LIFO) e Filas (FIFO).

Algoritmos: Métodos de Ordenação (como Bubble Sort, Insertion Sort, Quick Sort) e algoritmos de Busca (Binária e Sequencial).

Análise de Complexidade: Introdução à Notação Big O para avaliar o desempenho de tempo e espaço dos algoritmos desenvolvidos.

# English
# Data Structures I
The Data Structures I course is essential for developing efficient software and understanding how information is organized within computer memory. The curriculum focuses on the study of linear structures and complexity analysis.

Core Concepts: In-depth study of Pointers and Dynamic Memory Allocation.

Linear Structures: Implementation and manipulation of Linked Lists (singly and doubly), Stacks (LIFO), and Queues (FIFO).

Algorithms: Sorting methods (such as Bubble Sort, Insertion Sort, Quick Sort) and Searching algorithms (Binary and Sequential).

Complexity Analysis: Introduction to Big O Notation to evaluate the time and space performance of developed algorithms.
